# Johannes Brahms

Un videojuego educativo 2D sobre la vida de Johannes Brahms, con un estilo arcade clásico (píxeles) y banda sonora estilo 8/16 bits.

## Cómo jugar

1. Abre la carpeta `brahms-game` en un editor de código como Visual Studio Code.
2. Inicia un servidor local. Si usas la extensión "Live Server" en VS Code, solo haz clic derecho en `index.html` y selecciona **"Open with Live Server"**.
3. El juego se abrirá automáticamente en tu navegador web.

## Controles (General)
- **Flechas Izquierda / Derecha (? / ?)**: Moverse.
- **Espacio**: Saltar / Avanzar diálogos.
- **E**: Interactuar.
- **P**: Pausa.
